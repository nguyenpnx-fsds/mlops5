# Welcome to demo about the basic RAG pipeline

## Setup

To run this demo, please install uv via [docs](https://docs.astral.sh/uv/getting-started/installation/)

Then run,
```bash
cd demo && \
uv venv && \
source .venv/bin/activate && \
uv sync --active
```


## To run the basic RAG pipeline

```bash
uv run main.py
```
